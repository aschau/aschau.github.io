Starview

My last project at Trigger Global. Made in Unity 3D.

I was the main developer for this application with some support from two developers on implementing some of the user interface, graphics, and data.

This project was intended as a to visualize live stats for Dallas Home games in the AT&T stadium to showcase their 5G technology as well as give fans access to the latest play stats and supplementary data. The application was a module that was integrated from Unity into the Dalls Cowboys native application. I assisted in guiding the third party, Yinzcam, which was in charge of the native application through the integration process.

I created and maintained the base architecture for the application as well as managed all the necessary API calls and protocols necessary to receive the live data for a game.

All of the visual stats you can see in my Starview UI Walkthrough video below are from a past game while the touchdown video showcases the last play from an actual live game in the stadium.
