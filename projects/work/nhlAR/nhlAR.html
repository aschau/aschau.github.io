NHL AR

My favorite project at Trigger Global. Made in Unity 3D.

I was the main developer for this application with some support from two developers on implementing some of the user interface.

This project was intended as a prototype to showcase a data company named JOGMO which create tracking technology for sports companies to help manage live player statistical data. It was also partnered with Intel to showcase their MEC server system in the rink.

I created and maintained the base architecture for the application as well as managed all the necessary API calls and protocols necessary to receive the live data for a game.

All of the visual stats you can see in my Live Data Example video below are from an originally live game, including player movement, speeds, and play by plays. I'm also seen showcasing the application in a video in this tweet by NHL from timestamps 0:04 - 0:08 and 0:48 - 0:53.
