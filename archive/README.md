# Archive

Old versions and redesign explorations preserved for reference. Not part of the live site, not in the sitemap, not linked from the current pages.

## Contents

- **`index-old.html`** — The glassmorphism-era home page (v2). Full career timeline, sprite animations, particles. Was the live homepage through early 2026.
- **`index-arcade.html`** — Snapshot of the arcade home page before the SPA refactor split it into `index.html` + `sections/*.html`. Preserved by commit `a4c2056`.
- **`404-arcade.html`** — Old 404 page from the arcade preview. The current `404.html` is at repo root.
- **`redesign-preview.html`**, **`redesign-preview-2.html`**, **`redesign-preview-3.html`** — Three proof-of-concept redesign pages built in April 2026 to explore different directions before settling on the arcade concept. Preview 3 evolved into the current arcade.
- **`favicon-extractor.html`** — Dev utility for extracting favicons.

These files are still web-accessible (GitHub Pages serves them) if you want to visit `raggedydoc.com/archive/redesign-preview-3.html` etc. — useful for the "eras" cards in the Personal → Website deck.
