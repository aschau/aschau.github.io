How to Use

1
Import or Draw

Import a floorplan image or use the wall tool to draw your room.

2
Set Scale

Use the ruler tool to draw a line across a known distance, then enter the real measurement.

3
Add Furniture

Tap items from the bottom bar to place them. Drag to move, use handles to rotate and resize.

4
Navigate

Scroll to zoom, click & drag with the pan tool (or middle mouse) to move around.
