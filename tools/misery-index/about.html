What is this?

The Claude Developer Misery Index measures how much developers are suffering during Claude AI outages and degradations. It combines the official Anthropic status page, recent incident history, and real-time complaint volume from Reddit to produce a single score from 0 (bliss) to 10 (apocalypse).

Status pages are notoriously slow to update. By the time it officially says "Investigating," Reddit has already posted fifteen "is it just me?" threads. The Misery Index captures that gap — and the uncomfortable truth about how dependent we've all become on AI to write our code.

Data sources

Anthropic Status Page — the ground truth. Provides a system-level indicator (none, minor, major, critical) and per-component status for claude.ai, the API, Claude Code, and more.

Reddit — the human suffering layer. Searches for posts containing phrases like "Claude is down," "Claude outage," and "Claude not working" across r/ClaudeAI, r/anthropic, and the broader site. Post volume and comment depth both contribute to the score.

The status page carries the most weight because confirmed outages are the most reliable signal. Reddit volume is secondary — it's noisy, but it's fast. Comments amplify rather than dominate, because a single viral complaint thread isn't the same as twenty independent reports.

Misery levels

ALL CLEAR (0–1) Everything works. Suspicious.
MINOR GRUMBLING (1–3) A few Reddit posts. Probably fine.
GROWING UNREST (3–5) The subreddit is heating up.
FULL MELTDOWN (5–7) Confirmed outage. Devs remembering how to read docs.
APOCALYPSE (7–10) Everything is on fire. The screen shakes.

How data is collected

Data is refreshed every 15 minutes automatically. The dashboard also fetches live status and incident data directly, so it stays current between refreshes.

History is preserved for 7 days, displayed as a 48-hour rolling sparkline on the dashboard.

Limitations

This is a fun side project, not an SLA monitor. Reddit keyword matching is imperfect — some posts are about slowness rather than outages, and some genuine complaints use different phrasing. The 15-minute polling interval means short blips might be missed entirely.

The index also doesn't distinguish between "Claude is down for everyone" and "Claude is slow for some users." Both generate Reddit posts, but the severity is very different. Use the official status page for actual incident tracking.
